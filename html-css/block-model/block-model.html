Bir Elemanın Blok (Kutu) Modeli

Giriş

HTML ve CSS’de bir elemanın kutu modeli; içerik (content), iç boşluk (padding), kenarlık (border) ve dış boşluk (margin) alanlarını kapsar.

Blok Modeli Nedir? (Kutu Modeli Tanımı)

Bir elemanın genişlik ve yüksekliğini belirleyen düzen kuralıdır. Aşağıdaki unsurlar vardır:
  • Content: Asıl eleman içeriği. (Örn: metin, resim)
  • Padding: İçerik ile kenarlık arasındaki boşluk.
  • Border: Elemanın etrafını saran çizgi.
  • Margin: Eleman ile çevresindeki diğer öğeler arasındaki boşluk.
/* Örnek: Content-box varsayılan davranış */ /* Not: width yalnızca içerik alanını belirler */ .box { width: 150px; padding: 30px; /* içerikle border arasındaki boşluk */ border: 5px solid red; margin: 20px; }

Özelliklerin Detayları

Padding: Elemanın içeriği ile kenarlığı arasındaki boşluğu belirler. Örneğin, padding: 20px; elemanın tüm kenarlarında 20px boşluk oluşturur.

Border: Elemanın kenarlığını oluşturur. Örneğin, border: 2px dashed #333; kenarlık ekler.

Margin: Elemanın dışındaki boşluğu ayarlar. Varsayılan tarayıcı değerleri genellikle paragraflarda ve başlıklarda ek boşluk bırakır.

/* Varsayılan tarayıcı değerleri örneği */ /* h1, p gibi öğelerde genellikle margin eklenmiştir */ h1 { margin: 0.67em 0; } p { margin: 1em 0; }

Deneme Örneği

Alternatif Kullanımlar: border-box

Eğer box-sizing: border-box; kullanırsanız, width ve height tüm elemanı kapsar; padding ve border hesaplama dışında kalır.
/* Border-box örneği */ .box { box-sizing: border-box; width: 150px; padding: 30px; border: 5px solid red; margin: 20px; }

Border-box
 

Box-Sizing Özelliği

/* Global ayar */ *, *::before, *::after { box-sizing: border-box; }

Padding & Margin Kullanımı

Padding ve margin özelliği kullanılırken, kendi değerlerinizi belirleyebilirsiniz. Fakat unutmayın, tarayıcıların varsayılan stil sayfalarında bu değerlere otomatik eklemeler olabilir.
/* Örnek: padding: 10px 20px; margin: 0 0 15px 0; */

Esnek Görüntüler ve Object-Fit

Görüntüler, kapsayıcılarının genişliğine göre ayarlanır. object-fit: cover; oranları korur.
/* Görüntü esnekliği için */ img { display: block; max-width: 100%; object-fit: cover; }

Display Özelliği ve Öğelerin Türleri

Block, inline ve inline-block öğeler farklı davranışlar sergiler. Örneğin, inline öğelere width veya height uygulanmaz.
